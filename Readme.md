## Первый группойвой проект

Цель - сделать свою версию инструмента wc

Roadmap:
1. Ввод - запуск и название файла
    1.1 Проверить существует ли файл
        Если нет вывести ошибку
        Если есть вывести ехо названия
    результат: запуск, ввод, ехо названия или вывод ошибки при отсутствии файла, выход
2. Обработка флагов
    2.1.Если флагов нет вывести информацию по умолчанию
        Если флаги есть вывести необходимую информацию
    результат: запуск, ввод, вывод информации при отсутствии ошибок, выход

1. Прочитать ввод (до запуска программы)
    1.1 Флаги
2. Выполнить то что нужно (зависит от флагов)
    - help
    - error
    - default
    - -l -- lines
    - -w -- words
3. Вывести
